🎨 Assets Library

A personal collection of reusable assets including icons, logos, and UI elements used across multiple projects.

This repository is built to maintain design consistency, improve development speed, and keep all visual resources organized in one place.

```
│

```

🧠 Best Practices

- Use SVG for icons (better quality + smaller size)
- Optimize images before uploading
- Keep naming clean and consistent
  - "home.svg"
  - "home-active.svg"
  - "home-outline.svg"
